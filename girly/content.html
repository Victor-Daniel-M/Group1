The table below shows the course outline and links to contents

WEEKS

HOURS

CONTENT

INTENTED LEARNING OUTCOMES
Week 1 & 2 8 hours

Complexity analysis (Big-O notation, orders of growth, worst case, average case);
  1. Be able to understand and appreciate the role of data organization and algorithmic structure.
  2. Be able to compare the efficiency of a good programming algorithm.
  3. Argue the correctness of algorithms using inductive proofs and loop invariants
  4. Analyze worst-case running times of algorithms using asymptotic analysis
  5. Prove the correctness and analyse the running time of the basic algorithms for those classic problems in various domains
Week 3 4 hours

Recursive data structures: Merge, Quick sort
  1. Understand the mechanics of recursion.
  2. To implement recursive formulation of a problem structures.
  3. Understand how recursion is implemented by a computer system.
  4. Analysis of recursion.
Week 4 4 hours

Elementary Data Structures: Lists, Stack and Queues, Priority Queues
  1. Explain the concept of abstract data types (ADT)
  2. Understand the abstract data types; lists, stacks and queues
  3. Analyze the performance of the implementation of the linear basic data structures.
  4. Assess how the choice of data structures and algorithm design methods impacts the performance of programs.
  5. Develop algorithms for manipulating stacks, queues, linked lists, trees, graphs
  6. Analyze, implement, and use linked list, double-linked list, circular-linked list, and recursion on linked list
Week 5 4 hours

Sorting Algorithms: Bubble, Insertion selection, merge, and quick sort.
  1. Explain the major algorithms for sorting.
  2. Recite the analyses of these algorithms and the design strategies that the algorithms embody.
  3. Synthesize algorithms that employ sorting as a sub procedure.
  4. Derive lower bounds on the running time of comparison-sorting algorithms, and explain how these bounds can be overcome.
Week 6 4 hours

Searching Algorithms: Linear Search, Binary search
  1. Explain and implement sequential search and binary search.
  2. Be able to analyse the performance of these searching algorithms.
Week 7 & 8 8 hours

Trees and Tree algorithms: Binary tree applications, Tree Traversals, Binary search Trees

  1. Understand what a tree data structure is and how they are used.
  2. Implementation of trees using a list.
  3. Implementation of trees as a recursive data structure.
  4. Develop algorithms for manipulating, trees.
Week 9 & 10 8 hours

Graph and Graph Algorithms: Topological sort, Minimum spanning trees,Shortest path

  1. Explain the major graph algorithms and their analyses.
  2. Employ graphs to model engineering problems, when appropriate.
  3. Synthesize new graph algorithms and algorithms that employ graph computations as key components, and analyze them
Week 10 - 13 8 hours

Project :Course projects aimed at problem solving, design, and implementation
  1. Demonstrate a familiarity with applied algorithmic settings in real problem-solving
  2. Evaluation of Correctness, Design and Implementation, and Documentation of algorithms.
Week 14 - 17 8 hours

Examinations
